HTML Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.

HTML Comment Tags

You can add comments to your HTML code using the following syntax:

Example

<!-- This is a comment -->

Comments are helpful for explaining code, organizing sections, or leaving notes for other developers.

Add Comments

With comments, you can place reminders or explanations in your code without affecting the visual output.

Example

<!-- This is a comment -->
<p>This is a paragraph.</p>

The comment will not be visible in the browser.

Hide Content

You can also use comments to hide content that you don't want to be visible temporarily:

Example

<p>This is a visible paragraph.</p>
<!-- <p>This is a hidden paragraph.</p> -->

Only the first paragraph will be shown.

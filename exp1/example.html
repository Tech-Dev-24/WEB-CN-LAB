HTML <b> Tag

The <b> tag specifies bold text without any extra importance.

Example

Make some text bold (without marking it as important):

Example

<p>This is normal text - <b>and this is bold text</b>.</p>

Result:

This is normal text - and this is bold text.

Tips and Notes

Note: According to the HTML5 specification, the <b> tag should be used as a LAST resort when no other tag is more appropriate. The specification states that headings should be denoted with the <h1> to <h6> tags, emphasized text should be denoted with the <em> tag, important text should be denoted with the <strong> tag, and marked/highlighted text should use the <mark> tag.

Default CSS Settings

Most browsers will display the <b> element with the following default values:

b {

  font-weight: bold;

}

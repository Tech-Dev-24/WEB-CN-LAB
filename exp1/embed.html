HTML <embed> Tag

The <embed> tag defines a container for an external resource, such as a web page, a picture, a media player, or a plug-in application.

Example

An embedded Flash animation:

Example

<embed type="application/x-shockwave-flash" src="animation.swf" width="400" height="300">

Definition and Usage

The <embed> tag defines a container for an external resource.

External resources can be:

  • Web pages
  • Pictures
  • Media players
  • Plug-in applications

Note: The <embed> tag is supported in all major browsers, but the definition of what content should be embedded differs between browsers.

Attributes

Attribute Value Description
height pixels Specifies the height of the embedded content
src URL Specifies the address of the external file to embed
type media_type Specifies the media type of the embedded content
width pixels Specifies the width of the embedded content

Browser Support

Element Chrome Edge Firefox Safari Opera
<embed> Yes Yes Yes Yes Yes

Global Attributes

The <embed> tag supports the Global Attributes in HTML.

Event Attributes

The <embed> tag supports the Event Attributes in HTML.

Default CSS Settings

Most browsers will display the <embed> element with the following default values:

embed:focus { outline: none; }

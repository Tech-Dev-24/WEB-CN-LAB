HTML <a> Tag

The HTML <a> tag defines a hyperlink, which is used to link from one page to another.

Example

Here's a basic example of an anchor tag:

Example

<a href="https://www.w3schools.com">Visit W3Schools</a>

Output:

Visit W3Schools

Attributes

The most important attribute of the <a> tag is the href attribute, which indicates the link's destination.

Attribute Description
href Specifies the URL of the page the link goes to
target Specifies where to open the linked document (_blank = new tab)
title Specifies extra information (displayed as a tooltip on hover)
download Indicates that the target will be downloaded when clicked

More Examples

Example

<a href="page.html" target="_blank">Open in new tab</a>

Output:

Open in new tab

Browser Support

The <a> tag is supported in all major browsers.

Summary

The <a> tag creates a hyperlink. Use href to define the link's destination, and target for where to open it.

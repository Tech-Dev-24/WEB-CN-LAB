HTML <em> Tag

The <em> tag is used to define emphasized text. The content inside is typically displayed in italic.

A screen reader will pronounce the words in <em> with an emphasis, using verbal stress.

Example

Mark up emphasized text in a document:

Example

<p>You <em>have</em> to try this recipe!</p> <p>We <em>cannot</em> live like this.</p>

Result:

You have to try this recipe!

We cannot live like this.

Definition and Usage

The <em> tag is used to define emphasized text. The content inside is typically displayed in italic.

A screen reader will pronounce the words in <em> with an emphasis, using verbal stress.

Tip: Use the <em> tag when you want to add stress emphasis to a word or phrase.

Default CSS Settings

Most browsers will display the <em> element with the following default values:

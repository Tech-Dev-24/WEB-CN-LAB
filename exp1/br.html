HTML <br> Tag

The <br> tag inserts a single line break.

The <br> tag is useful for writing addresses or poems.

The <br> tag is an empty tag which means that it has no end tag.

Example

Use the <br> tag to insert line breaks in a text:

Example

<p>To force<br> line breaks<br> in a text,<br> use the br<br> element.</p>

Definition and Usage

The <br> tag inserts a single line break.

The <br> tag is an empty tag which means that it has no end tag.

Tips and Notes

Tip: The <br> tag is useful for writing addresses or poems.

Note: Use the <br> tag to enter line breaks, not to add space between paragraphs.

Default CSS Settings

Most browsers will display the <br> element with the following default values:

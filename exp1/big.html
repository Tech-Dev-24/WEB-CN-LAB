HTML <big> Tag

Not Supported in HTML5.

The <big> tag was used in HTML 4 to define bigger text.

What to Use Instead?

Example

Use CSS to set larger font size:

<p>This is <span style="font-size: larger;">larger text</span>.</p>

Result:

This is larger text.

Definition and Usage

The <big> tag was used to define bigger text.

Tip: Use CSS instead (see example above).

Browser Support

Element Chrome Edge Firefox Safari Opera
<big> No No No No No

Global Attributes

The <big> tag does not support any standard attributes in HTML5.

Event Attributes

The <big> tag does not support any event attributes in HTML5.

Default CSS Settings

Most browsers will display the <big> element with the following default values:

big { font-size: larger; }

HTML <center> Tag

Not Supported in HTML5.

The <center> tag was used in HTML4 to center-align text.

What to Use Instead?

Example

Use CSS instead (with either the text-align property or margin: auto for block elements):

<!-- For text content --> <div style="text-align: center;"> This text will be centered </div> <!-- For block elements --> <div style="width: 50%; margin: auto;"> This div will be centered </div>

Result:

This text will be centered

This div will be centered

Definition and Usage

The <center> tag was used to center-align text in HTML4.

This element is obsolete in HTML5 and should not be used.

Browser Support

Element Chrome Edge Firefox Safari Opera
<center> No No No No No

Global Attributes

The <center> tag does not support any standard attributes in HTML5.

Event Attributes

The <center> tag does not support any event attributes in HTML5.

Default CSS Settings

Most browsers will display the <center> element with the following default values:

center { text-align: center; }

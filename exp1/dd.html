HTML <dd> Tag

The <dd> tag is used to describe a term/name in a description list.

The <dd> tag is used in conjunction with <dl> (defines the description list) and <dt> (defines terms/names).

Example

A description list, with terms and descriptions:

Example

<dl> <dt>Coffee</dt> <dd>Black hot drink</dd> <dt>Milk</dt> <dd>White cold drink</dd> </dl>

Result:

Coffee
Black hot drink
Milk
White cold drink

Definition and Usage

The <dd> tag is used to describe a term/name in a description list.

The <dd> tag is used in conjunction with <dl> (defines a description list) and <dt> (defines terms/names).

Inside a <dd> tag you can put paragraphs, line breaks, images, links, lists, etc.

Browser Support

Element Chrome Edge Firefox Safari Opera
<dd> Yes Yes Yes Yes Yes

Global Attributes

The <dd> tag also supports the Global Attributes in HTML.

Event Attributes

The <dd> tag also supports the Event Attributes in HTML.

Default CSS Settings

Most browsers will display the <dd> element with the following default values:

dd { display: block; margin-left: 40px; }

HTML <dt> Tag

The <dt> tag defines a term/name in a description list.

The <dt> tag is used in conjunction with <dl> (defines the description list) and <dd> (describes each term/name).

Example

A description list, with terms and descriptions:

Example

<dl> <dt>Coffee</dt> <dd>Black hot drink</dd> <dt>Milk</dt> <dd>White cold drink</dd> </dl>

Result:

Coffee
Black hot drink
Milk
White cold drink

Definition and Usage

The <dt> tag defines a term/name in a description list.

The <dt> tag is used in conjunction with <dl> (defines a description list) and <dd> (describes each term/name).

Inside a <dt> tag you can put paragraphs, line breaks, images, links, lists, etc.

Browser Support

Element Chrome Edge Firefox Safari Opera

Global Attributes

The <dt> tag supports the Global Attributes in HTML.

Event Attributes

The <dt> tag supports the Event Attributes in HTML.

Default CSS Settings

Most browsers will display the <dt> element with the following default values:

dt { display: block; }

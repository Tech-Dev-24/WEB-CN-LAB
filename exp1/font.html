HTML <font> Tag

Not Supported in HTML5.

The <font> tag was used in HTML4 to specify the font face, font size, and color of text.

What to Use Instead?

Example

Use CSS to style text:

<p style="font-family: Arial; color: #FF0000; font-size: 20px;"> This is styled text using CSS </p>

Result:

This is styled text using CSS

Definition and Usage

The <font> tag was used in HTML4 to specify the font face, font size, and color of text.

This element is obsolete in HTML5 and should not be used. Use CSS instead.

Attributes

Attribute Value Description
color color Specifies the color of text
face font_family Specifies the font of text
size number Specifies the size of text (1-7)

Browser Support

Element Chrome Edge Firefox Safari Opera
<font> No No No No No

Global Attributes

The <font> tag does not support any standard attributes in HTML5.

Event Attributes

The <font> tag does not support any event attributes in HTML5.

Default CSS Settings

Most browsers will display the <font> element with the following default values:

font { font-family: serif; font-size: medium; font-style: normal; font-weight: normal; color: #000000; }

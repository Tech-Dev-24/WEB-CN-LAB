HTML <dl> Tag

The <dl> tag defines a description list.

The <dl> tag is used in conjunction with <dt> (defines terms/names) and <dd> (describes each term/name).

Example

A description list, with terms and descriptions:

Example

<dl> <dt>Coffee</dt> <dd>- black hot drink</dd> <dt>Milk</dt> <dd>- white cold drink</dd> </dl>

Result:

Coffee
- black hot drink
Milk
- white cold drink

Definition and Usage

The <dl> tag defines a description list.

The <dl> tag is used in conjunction with:

  • <dt> - Defines terms/names
  • <dd> - Describes each term/name

Browser Support

Element Chrome Edge Firefox Safari Opera
<dl> Yes Yes Yes Yes Yes

Global Attributes

The <dl> tag also supports the Global Attributes in HTML.

Event Attributes

The <dl> tag also supports the Event Attributes in HTML.

Default CSS Settings

Most browsers will display the <dl> element with the following default values:

dl { display: block; margin-top: 1em; margin-bottom: 1em; margin-left: 0; margin-right: 0; }

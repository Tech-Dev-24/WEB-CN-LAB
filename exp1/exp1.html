HTML Tags Explorer

A beginner's guide to essential HTML tags with live examples

<!DOCTYPE html>

The document type declaration. It tells the browser this is an HTML5 document.

<!DOCTYPE html>
Output: (Not visible - this is the first line of an HTML document)

<html>

The root element of an HTML page. All other elements are contained within this.

<html lang="en">...</html>
Output: (This tag wraps all content on the page)

<head>

Contains meta information about the document, like its title, styles, and scripts.

<head>
  <title>My Page</title>
</head>
Output: (Not visible - contains metadata and links to resources)

<body>

Contains the visible page content - text, images, links, etc.

<body>
  <h1>Hello World!</h1>
</body>
Output: (All visible content on this page is within the body tag)

<h1> to <h6>

Heading elements, where h1 is the most important and h6 the least.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>
Output:

Main Heading

Sub-heading

Sub-heading

Sub-heading

Sub-heading
Sub-heading

<p>

Defines a paragraph of text.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
Output:

This is a paragraph.

This is another paragraph.

<br>

Inserts a single line break (carriage return).

First line<br>
Second line
Output:
First line
Second line

<hr>

Creates a thematic break (horizontal rule) between sections.

<p>First section</p>
<hr>
<p>Second section</p>
Output:

First section

Second section

<strong> and <b>

<strong> indicates important text (semantic), while <b> simply makes text bold (presentational).

This is <strong>important</strong> text.
This is <b>bold</b> text.
Output:
This is important text.
This is bold text.

<em> and <i>

<em> indicates emphasized text (semantic), while <i> simply makes text italic (presentational).

This is <em>emphasized</em> text.
This is <i>italic</i> text.
Output:
This is emphasized text.
This is italic text.

<mark>

Defines text that should be marked or highlighted.

This is <mark>highlighted</mark> text.
Output:
This is highlighted text.

<a>

Defines a hyperlink to another page or location.

<a href="https://example.com">Visit Example</a>
Output:
Visit Example

Common Attributes:

  • href: The URL the link points to
  • target: Where to open the link (_blank for new tab)
  • title: Additional information about the link

<nav>

Defines a section of navigation links.

<nav>
  <a href="/home">Home</a>
  <a href="/about">About</a>
  <a href="/contact">Contact</a>
</nav>
Output:
Home | About | Contact

<ul>, <ol>, and <li>

<ul> creates an unordered (bulleted) list, <ol> creates an ordered (numbered) list, and <li> defines list items.

<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

<ol>
  <li>First step</li>
  <li>Second step</li>
</ol>
Output:
  • First item
  • Second item
  1. First step
  2. Second step

<dl>, <dt>, and <dd>

Creates a description list. <dt> defines the term, and <dd> describes it.

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
Output:
HTML
HyperText Markup Language
CSS
Cascading Style Sheets

<img>

Embeds an image in the document.

<img src="https://via.placeholder.com/150" alt="Placeholder image" width="150" height="150">
Output:

Common Attributes:

  • src: The image URL
  • alt: Alternative text for accessibility
  • width/height: Image dimensions

<audio>

Embeds sound content.

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>
Output:

Note: Audio file not included in this example

<video>

Embeds video content.

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>
Output:

Note: Video file not included in this example

<table>, <tr>, <th>, <td>

Creates a table. <tr> defines a table row, <th> defines a header cell, and <td> defines a standard cell.

<table>
  <tr>
    <th>Month</th>
    <th>Savings</th>
  </tr>
  <tr>
    <td>January</td>
    <td>$100</td>
  </tr>
</table>
Output:
Month Savings
January $100

<form>

Creates an HTML form for user input.

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>
Output:

<input>

Creates an input field. The type attribute determines what kind of input.

<input type="text" placeholder="Text input">
<input type="password" placeholder="Password">
<input type="checkbox" id="check1"> <label for="check1">Checkbox</label>
<input type="radio" id="radio1" name="radio"> <label for="radio1">Radio 1</label>
<input type="submit" value="Submit">
Output:

Creates a multi-line text input control.

<textarea rows="4" cols="50">
Default text goes here...
</textarea>
Output:

<select>, <option>

Creates a drop-down list.

<select>
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="mercedes">Mercedes</option>
</select>
Output:

<button>

Creates a clickable button.

<button type="button">Click Me!</button>
Output:

<header>

Represents introductory content, typically a group of introductory or navigational aids.

<header>
  <h1>Website Title</h1>
  <p>Website slogan</p>
</header>
Output:

Website Title

Website slogan

<footer>

Represents a footer for its nearest sectioning content or root element.

<footer>
  <p>© 2023 My Website</p>
</footer>
Output:

© 2023 My Website

<section>

Defines a section in a document.

<section>
  <h2>Section Title</h2>
  <p>Section content...</p>
</section>
Output:

Section Title

Section content...

<article>

Specifies independent, self-contained content.

<article>
  <h2>Article Title</h2>
  <p>Article content...</p>
</article>
Output:

Article Title

Article content...

<aside>

Defines content aside from the content it is placed in (like a sidebar).

<aside>
  <h3>Related Links</h3>
  <ul>
    <li><a href="#">Link 1</a></li>
    <li><a href="#">Link 2</a></li>
  </ul>
</aside>
Output:

Related Links

  • Link 1
  • Link 2

<div>

A generic container for flow content with no semantic meaning.

<div class="container" style="border: 1px solid #000; padding: 10px;">
  <p>Some content here</p>
</div>
Output:

Some content here

<span>

A generic inline container for phrasing content with no semantic meaning.

<p>This is <span style="color: red;">red text</span>.</p>
Output:

This is red text.

<code>

Displays its contents styled in a fashion intended to indicate computer code.

<code>const x = 5;</code>
Output: const x = 5;

Basic Knowledge of Web Development

Frontend Development

Frontend development involves creating the user interface and user experience of a website. It primarily uses three core technologies:

  • HTML (HyperText Markup Language) - The structure of web pages
  • CSS (Cascading Style Sheets) - The styling and layout
  • JavaScript - The behavior and interactivity

Modern frontend development often uses frameworks like React, Angular, or Vue.js to build complex applications.

Backend Development

Backend development handles the server-side logic, databases, and application functionality. Common backend technologies include:

  • Server-side languages like Node.js, Python, PHP, Ruby, or Java
  • Databases like MySQL, PostgreSQL, or MongoDB
  • Web servers like Apache or Nginx

Backend frameworks such as Express (Node.js), Django (Python), or Laravel (PHP) help streamline development.
